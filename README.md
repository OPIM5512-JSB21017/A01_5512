## A01

### Assignment Overview:
In this assignment, we are learning how to:
- Use **GitHub** for version control and project organization  
- Work with the **California Housing dataset**  
- Generate and save a simple visualization using Python  

---

### How to Run the Script:

1. Make sure **Python 3.12.x** is installed on your machine  
2. Open the repository in **VS Code**  
3. Open a terminal in the project root folder  
4. Install the required packages:
   ```bash
   python -m pip install -r requirements.txt
   ```
5. Run the script
   ```bash
   python src/boxplot.py
   ```

--- 
### Expected Output:
- A preview of the California Housing dataset and its shape printed to the console
- A boxplot of Median House Value generated using matplotlib
- The boxplot is saved to:
   - figs/boxplot.png
