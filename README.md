## A01

### - In this assignment, we're learning how to: Utilize github. 
### - We are using a california housing dataset
### - How to run the script:

1. Make sure Python 3.12.x is installed on your machine.
2. Open the repository in VS Code.
3. Open a terminal in the project folder.
4. Install the required packages by running:
   
   python -m pip install -r requirements.txt
   
5. Run the script with:
   
   python src/boxplot.py

### - Expected output:
- The script prints a preview of the California Housing dataset and its shape.
- A boxplot of Median House Value is generated.
- The boxplot is saved as: figs/boxplot.png
